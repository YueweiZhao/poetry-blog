# 观敦煌莫高窟

驼铃丝路到敦煌，彩绘千年观汉唐。

历史长河留瑰宝 ，文明华夏屹东方。

# 敦煌鸣沙山见闻

# 一

茫茫沙海浪涛狂，队队驼铃向远方。

古道西风今又起，中华丝路万年昌。

# 二

中华通韵

奇观大漠月牙泉，漫漫黄沙泛碧涟。

亘古澄澄尘不染，瑶池何故落人间？

# 张掖丹霞地质公园

叠嶂层峦腾七彩，磅礴气势触心惊。

何方神笔留神韵，飘落陇原留绮情。

# 张掖平山湖大峡谷

# 一

五彩斑斓混沌开，层峦叠嶂是谁裁？

神奇造化摄心魄，画卷绵绵寄梦台。

# 二

河西张掖有遗篇，气势恢宏成大川。

沟壑纵横千万万，峰林奇特万千千。

金蟾翘首观天宇，汉将回眸眺玉关。

鬼斧神工惊眼目，苍凉壮美扣心弦。

注："玉门"指玉门关。此处有一地貌象汉朝霍去病将军披甲远眺。

# 夏游天山天池

行至瑶池碧水前，波平如鉴半空悬。

远山銜雪微澜映，松韵氤氲系客船。
