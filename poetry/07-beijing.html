# 游紫竹院

紫竹一庭荷满塘，游人驻足赏风光。

微风掠过清香溢，薄雾飘来爽气凉。

远处红歌声响亮，近旁琴曲韵悠扬。

抬头喜看双飞燕，逐日追云自在翔。

# 植物园万生苑一隅

岚烟漫碧台，似梦到蓬莱。

花族瑶池满，佳人笑靥开。

# 登北京后花园

古稀执杖去登山，坡险路岐何畏艰。

冰瀑蜿蜒凌上过，回眸来处展眉颜。

注：北京后花园景区属昌平区阳坊镇境内，属太行余脉。

# 景山公园赏芍药

可怜芍药此园逢，灿若云霞蜂蝶惊。

丽质天生娇妩媚，随风起舞袅婷婷。

# 观故宫贺岁迎祥清朝宫廷年俗展

紫金城中年味展，近千珍品一丛观。

明窗开笔真容见，祈福岁丰天下安。

注：2019年春节故宫贺岁迎祥宫廷年俗展，清朝从雍正朝开始，元旦皇帝开笔均写吉语，写好后即封藏起来，不许开看，是默默地祝福。今有机会放见真容。

# 动物园观水禽湖

中华通韵

水禽湖畔水禽喧，众鸟纷纷一乐园。

嬉戏追逐珠玉溅，倏然展翅入蓝天。

# 春游颐和园

十里葱茏万寿山，西堤垂柳自清闲。

昆明湖畔几多事，一代帝王谈笑间。

# 游圆明园有感

大清愚弱列强欺，断壁残垣满创痍。

杨柳不知仇耻恨，临风依旧舞腰肢。

# 卢沟桥观后感

卢沟桥上弹痕藏，倭寇当年战火伤。

复兴中华强世界，岂容鬼子再猖狂。

# 参观"中国抗日纪念馆"

卢沟桥畔枪声响，华夏苍生遭重创。

凌辱奸淫人性丧，烧焚掠杀兽心狂。

军民奋起拼征战，倭寇仓逃溃败亡。

浴火重生强国梦，凤凰振翅傲东方。

# 观宛平城雄狮雕塑

雄狮觉醒仰天啸，嘶吼如雷震四方。

七十年前民族恨，时时警惕野心狼。

# 观九·三阅兵有感(2016)

参阅老兵倍感亲，枪林弹雨几人存。

舍身忘死驱倭寇，动地惊天泣鬼魂。

七七"硝烟留国耻，"九三"方阵亮军魂。

喜看装备惊天下，谁敢欺侵我国门。

注：九·三阅兵是2015.9.3在天安门地区纪念中国人民抗日战争暨世界反法西斯战争胜利70周年阅兵式。

# 祖国七十华诞颂

旗彩飘飘华诞迎，春秋七十正鹏程。

巡航母舰环沧海，展翅歼击耀美名。

西气东输民福祉，南流北调业兴荣。

江山锦绣昭天下，社稷繁荣惠众生。

两弹一星惊世界，三军千阵亮京城。

台湾港澳同欢庆，迈向未来肩并行。

# 中国共产党建党一百周年抒怀

中华通韵

马列先锋结众贤，丹心志士誓言宣。

红船驾驭航灯点，民庶跟随险浪穿。

推到三山寰宇镇，完成四化举国欢。

卫星登月太空探，舰艇劈波疆域安。

建党百年歌盛世，和风万里满人间。

沧桑巨变怀旗手，引领中华永向前。

# 忆总理

雪落神州天地忧，追思总理泪先流。

终身奋斗为华夏，伟绩丰功万世讴。

注：写于2019年冬季。

# 喜迎十九大

雨过天晴村野忙，金黄稻谷遍山乡。

喜迎盛会传高产，科技兴农奔小康。

注：中国共产党第十九次代表大会于2017.10在北京召开。在乘车途中观高速路沿岸一派丰收景象，随吟之。
