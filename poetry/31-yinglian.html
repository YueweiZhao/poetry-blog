  • 三十二、诗钟

晨晖收露；夜月铺霜。

一川烟草；十里荷花。

清风送爽；淡月笼明。

山花烂漫；彩玉斑斓。

风留竹韵；雪送梅香。

闻名即醉；回味犹香。

观天下事；读圣贤书。

春暖百花艳；秋凉万果香。

在窗前绘画；向院内观花。

春燕梁间绕；秋鸿云里翔。

和风生翠幕；疏雨落红荷。

蝉鸣柳树上；蝶舞花草间。

夜静蛩声脆；骄阳知了狂。

春风舒绿柳；夏露润红荷。

风竹流清韵；雨荷传雅音。

雁向南疆飞去；船从北海驶来。

曲径红英铺毡；满园白絮飘棉。

村晚炊烟升起；林深群鸟和鸣。

朝旭轻收叶露；晚风淡散花香。

簌簌雪飘原野；呼呼风过山梁。

姐共妹来练舞；嫂同姑去唱歌。

唉！人心难料；噢！世事无常。

夕阳西坠苍山静；夜月东升湖面明。

一川烟草一川雨；十里残荷十里霜。

雨染桃红霞万岭；风催柳绿树千行。

花开庭院香扑鼻；琴响厅堂韵绕梁。

绿杨影里吟唐句；黄果林中唱旧词。

五月榴花红似火；三秋霜菊白如云。

紫竹红芍庭院里；白云青岫画图中。

紫竹庭前琴韵荡；绿杨湖畔舞姿飞。

黄菊千层幽径绕；丹枫万叶碧空飘。

漠漠凉天闻征雁；潇潇冷雨打残荷。

泼墨挥毫抒雅兴；吟诗作画赋闲情。

种植花草呈闲趣；游览山川释逸情。

扶弱助贫行善举；排忧解难渡慈航。

恶言一句听真冷；秽语半声闻更烦。

燃炮燃烟辞旧岁；载歌载舞庆新春。

# 游渔沟天一园撰联

集纳江南园林秀雅；展扬灵璧石苑精神。

精通古今方能霸业；酷爱诗书可领风骚。

饱览群书可成大器；勤挥拙笔能著华章。

淡泊名利，可作闲云野鹤；贪得钱财，终成法网囚徒。

# 汴河灵璧段橡皮坝联

一朝完竣橡皮坝；万载丰收农民家。

# 同学婚联

并枝开并蒂；同牖结同心。

# 通用婚联

风和日丽门庭喜；花好月圆帷帐新。

# 赞才女联

芙蓉出水婵娟貌；文采惊人子建才。

瑶池不二，千秋绝唱销魂句；

紫府无双，万古兰章荡魄词。

# 狗年春联

鸡别旧年传喜讯；狗迎新岁送春晖。

# 鸡年春联

阍犬守门宅室泰；雄鸡报晓曙光升。

# 同音字联

田里甜瓜甜水溢；路边露草露珠莹。

# 重字联

春风春雨春花艳；秋露秋霜秋蕊寒。

# 对上联

宿云似幕能遮月；晚露如饴可润花。

# 对下联

溪边杨柳迎风舞；夹岸桃花蘸水开。
