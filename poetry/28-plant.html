# 迎春花

游园忽见一花开，顿觉春光扑面来。

串串金英摇翠蔓，新装新巧是谁裁？

# 杏花

# 一

山岭杏花茫，冰绡裁素装。

柔风香缕缕，仙子舞春光。

# 二

梅尽杏花开，幽香扑鼻来。

东风吹作雪，片片入尘埃。

# 桃花

隔岸桃花蘸水开，波光潋滟映红腮。

落英点点随流水，四季轮回君莫哀。

# 李花

三春已半景明时，竹外李花三两枝。

娇小琼英如白雪，婷婷袅袅弄仙姿。

# 芸薹

几经风雪几经霜，春雨催生遍野黄。

村后村前香韵绕，蜂飞蝶舞采花忙。

# 雨中海棠

三月海棠衔露开，宛如出浴贵妃来。

胭脂点点香腮艳，衒嫁东风莫费猜。

# 蒲公英

生来不与百花争，甘愿荒郊享静清。

只待一时丰羽翼，随风飘逸自躬耕。

# 小草

春雨春风破土生，纵横田陌草青青。

行人车辇相迎送，一片芳心一片情。

# 梨花

# 一

琼蕊漫枝丫，清芬远万家。

冰蕤迎远客，笑靥映丹霞。

# 二

中华通韵

玉树琼葩香雪海，东君着意著冰魂。

芳姿洁雅流幽韵，一首拈来和月吟。

# 三

春风吹雪舞轻柔，片片落花裙袂留。

少女娇羞遮欲露，梨园香韵意悠悠。

# 四

春风今日展鸿威，万树落英如雪飞。

花影丽人相映美，斜阳舞醉不思归。

# 月季

月季西园衔露开，娇容含笑送香腮。

芳馨熏得诗魂醉，袅袅吟声满梦台。

# 野蔷薇

腰软倚篱笆，丹唇灿若霞。

风中摇倩影，娇媚惹人夸。

# 红樱花

碧空如洗衬琼葩，粉面红腮染彩霞。

纵是人间三月暮，依然炫目靓春华。

# 栀子花

# 一

小径幽幽栀子花，冰魂雪魄皓无暇。

馨香缭绕牵衣袖，梦里一枝鬓发插。

# 二

轻采一枝五月花，冰姿香靥好清嘉。

孤芳自赏自陶醉，闲寄幽情伴月华。

# 石榴花

百芳凋谢随流水，五月榴花照眼明。

一片芳心何处诉？满腔珠玉只为情。

# 牡丹

牡丹朵朵艳阳开，风送馨香牵蝶来。

素蕊粉容娇艳艳，依稀仙子下瑶台。

# 雏菊

夏日炎炎雏菊放，风光不与四时同。

层层花木随风舞，绚烂多姿小院东。

# 红荷

# 一

轻舟远棹来，菡萏映桃腮。

十里平铺锦，红笺香韵裁。

# 二

红荷静静水中央，清雅恬然淡淡香。

偶起晨风掀翠盖，娇颜秀媚舞霓裳。

# 雨荷

# 一

阵阵斜风摇翠盖，绵绵细雨落清荷。

粉颜滴露舞婀娜，罗伞移舟闻棹歌。

# 二

翠盖风翻滚玉珠，平湖十里舞仙姝。

霏霏烟雨移轻棹，罗伞红裙入画图。

# 白荷

仙子素装凌步来，琼姿冷艳舞瑶台。

出泥不染留高洁，吻蕊蜜蜂香韵裁。

# 绿荷

惊艳绿荷花，莲池一丽葩。

红蕖羞掩面，朵朵笼蝉纱。

# 孤荷

一支独秀映池塘，碧水悠悠心事藏。

隔岸何人清笛韵，梅花三弄断柔肠。

# 并蒂莲

相慕相依并蒂莲，艳葩齐放展娇颜。

宛如出浴瑶池女，粉面香飘羞玉环。

# 秋荷

一枝静雅小荷塘，秋至依然呈靓装。

绰约风姿留晚韵，清风摇曳满池香。

# 残荷

碧叶粉颜曾艳阳，秋风秋雨陨红妆。

清魂入土芳菲尽，期待来年映日香。

# 金桂

芳园馥郁香，晓月挂梢旁。

一树宁宁立，叶藏金屑黄。

# 秋菊

# 一

临街一小园，三两傍篱边。

寒蕊清香溢，凌霜独傲然。

# 二

廊下阶前九月花，芳丛满目竞琼华。

伴云和月疏篱畔，湿露凝香是哪家？

# 寻梅

冰花朵朵扬，幽境觅幽香。

竹外琼枝瘦，梅开雪下藏。

# 腊梅

# 一

冰天雪地傲严寒，独秀一枝自在妍。

蜡染凝脂香气远，报知春汛近家园。

# 二

瘦枝寒蕊傲长空，三九鲜妍风雪中。

曼舞仙姿香韵洒，年年芳信约春风。

# 赠远

腊梅开岁寒，冷艳可人怜。

赠远芳盈手，暗香千里传。

# 咏梅

窗外几株梅蕊开，冰天雪地映楼台。

香魂缕缕红笺绕，点点寒英诗韵裁。

# 白梅

中华通韵

排排云雁来，江畔玉梅开。

一阵香风过，琼英落满怀。

# 红梅

春风吹梦到窗边，昨夜琼苞自騁妍。

素蕊粉腮芳信至，三分春色到庭前。

# 岁寒三友

# 一

松声梅影动，竹外瘦枝横。

牵手朔风里，冰寒结友情。

# 二

隆冬数九北风寒，松竹腊梅俱傲然。

窗外飘来香郁郁，频添雅韵贺新年。

# 君子兰

中华通韵

恬静娇莹君子兰，吐华展叶斗冬寒。

端庄潇洒芳悠远，惹袖牵衣伴梦眠。

# 蝴蝶兰

# 一

五彩斑斓蝴蝶兰，芳姿丽影舞翩跹。

盈盈绰态蛾眉展，一抹柔情尘世缘。

# 二

中华通韵

深山幽谷僻幽兰，细叶青青映素紈。

冷露清风香自远，花中君子惹人怜。

# 水仙花

凌波仙子素霓裳，水上轻盈玉露香。

不染尘埃清韵绝，生来高洁世无旁。

# 白山茶花

一株独秀仙葩靓，碧叶琼枝缀素妆。

偷去梨花三月白，飘来梅蕊九冬香。

# 竹

萧杀秋风催不垮，任由狂雪坠枝丫。

虚心向上昂然立，临变无惊气自华。
