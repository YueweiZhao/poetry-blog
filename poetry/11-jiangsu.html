# 登临北固山观景楼

秋风猎猎大江东，观景楼迎旭日红。

铁塔千年崖上立，谁人敢说不英雄。

注：铁塔是北固山的重要文物，我国仅存的六座铁塔之一。

# 北固山甘露寺

北固山幽禅意深，梵音袅袅气森森。

当年甘露结连理，千古风流说到今。

注：三国时甘露寺是孙刘联姻之地。

# 畅游焦山

烟波浩渺游帆远，林木森森郁郁然。

定慧梵音萦海岛，华严皓月洒江天。

山门松影摇佳客，庵院槐荫隐坐禅。

遥望涛涛东逝水，谁知能否断尘缘。

"定慧"即定慧寺，"华严"即华严阁，各为焦山十六景之一。

# 夜游泰州凤城河

画船轻荡晚风柔，亮透亭台水榭楼。

何处飘来天籁曲，韶音仿佛梦中游。

注：凤城河畔入夜，亭台水榭灯火通明歌舞笙平。

# 赏大丰郁金香

琼英遍野郁金香，千亩大丰千种装。

芳阵牵衣留雅客，流连花海忘还乡。

# 游丹顶鹤湿地有感

一片茫茫沼泽洼，娟姑寻鹤未回家。

芳魂追伴仙魂去，采采蒹葭不见花。

注：我国第一位驯鹤姑娘徐秀娟，1987年为救一只受伤的丹顶鹤，划进了沼泽地就再也没上来，牺牲了年轻的生命。

# 南山竹海

# 一

竹径深深缓缓行，春秋冬夏郁青菁。

虚心向上凌云志，风起篁喧惊鸟鸣。

# 二

竹香伴我踏清幽，岭岭修篁翠欲流。

泉水涓涓环曲径，争鸣禽鸟唱欢柔。

# 兴化采风

# 一

苏中二月采春光，遍地流金油菜黄。

娇艳芳馨开妩媚，游人花海踏花香。

# 二．李中水上公园

云遮雾盖碧流潺，满目芬芳二月兰。

闲步杉林闻鸟语，幽情洋溢泛心澜。

# 三．归途

青山绵不断，碧水润心田。

春踏江南梦，芳馨伴我还。

# 沙家浜荡舟

古镇度重阳，舟行芦苇塘。

忽闻杨柳调，俊俏一船娘。

# 夜秦淮河荡舟

# 一

十里秦淮不夜天，繁花点点柳如烟。

清歌流韵窗前水，明月邀侬上画船。

# 二

舟若游龙水里穿，流如缎带绕城间。

岸边绿柳波心月，乐度良宵笑语还。
