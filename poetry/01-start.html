# 课堂

霜鬓良师讲台立，摘词撷句解诗魂。

满堂华发神专注，声韵仄平求入门。

# 老年大学一隅

白首满堂君，仄平渐入门。

夕阳无限好，雅意染诗魂。

# 晨曲

中华通韵（限韵绝句：帘、蝉、淹）

晨光一缕透纱帘，高树频传饮露蝉。

篱下花丛蜂蝶舞，忽闻瑶曲雅心淹。

# 人生

（限韵律词：云、闻、焚、欣、勤）

人生百味众纷云，雅客庭前长见闻。

旭日东升朝露散，夕阳西下晚霞焚。

无情岁月催人老，有趣诗书慰尔欣。

年度古稀增逸兴，春秋冬夏四时勤。

# 辘轳体续绝四首

（"岁月总无情"各在1、2、3、4句）

# 一

岁月总无情，丝丝白发生。

初心仍未改，身老笔勤耕。

# 二

双栖双比翼，岁月总无情。

冥冥泉台隔，人生如梦轻。

# 三

落英铺小路，飞絮沾衣裤。

岁月总无情，留春春不驻。

# 四

往事履回萦，校园东草坪。

青春踪迹远，岁月总无情。

# 寿新元老师《散吟十三章》

# 一

一人头顶一爿天，当尽情怀享自然。

放胆勿辞游壑岭，洗心多去泡温泉。

怡辉腕底三分笔，悦弄庭旁半亩园。

且待喧嚣随夜逝，枕松抱月傍花眠。

# 二

琼楼未羡爱清庐，车马何关是有无。

闲看庭前花绽谢，漫观霜后叶沉浮。

德栽怀获玉千顷，情注诗凝冰一壶。

莫说风流儒雅事，人间烟火一凡夫。

# 三

涂鸦卌载艺难精，忝教一方多抱情。

匾额标牌悬店铺，楹联条幅挂堂厅。

殿高如索须三让，屋矮凡求只一声。

相悦两情休论价，金钱应信比人轻。

# 四

晋三仗势好张狂，军国灰燃修宪章。

欲掩侵行删铁证，思消屠罪吐雌黄。

深仇大耻当铭刻，强域固邦须奋扬。

用处莫显身已朽，根根老骨响当当。

# 五

浮生一梦过稀龄，两鬓霜侵不复青。

筋骨消磨如石铁，额眉淬砺铸锋棱。

趣来墨染千层锦，兴起笺流万点星。

老教诗台燃烛照，啸风驾雨走雷霆。

# 六

莫问残红为孰落，春光无奈漫蹉跎。

东风沉寂蝉争唱，寒雪纷扬莺不歌。

梅立冰崖魂傲世，松迎霜剑气吞河。

人生至老心无愧，鹤入云天自在哦。

# 七

岁晚喜吟归来去，髫年恨未识陶怀。

青云思步笑心蠢，黑洞不叹欣性呆。

斟句曾成三类货，出言多是一场灾。

身闲幸远沙尘路，自种桃花把酒猜。

注：三类货，指文革十六条中把社会人分成好的、比较好的、有严重错误和反党反社会主义右派四类。

# 八

东风起处又乘槎，采罢晨霞采晚霞。

诗海泛波辉朗月，心田种玉绽香葩。

不留污浊侵肌体，当让芬芳流岁华。

鹤岛梅林相约处，清清淡淡一杯茶。

# 九

春到芳庭眉眼舒，深红浅紫竞相涂。

牡丹斗艳梅先捷，月季流香兰未输。

虫损忙提喷药筒，土干即举洒泉壶。

呢喃紫燕深知意，筑巢西梁两不孤。

# 十

春宵都说值千金，年过古稀犹感深。

虽少裂开撕地力，尚存忧国恤民心。

剑锋仍横虎龙气，琴韵还流珠玉音。

常学叶公青岭眺，倚楼亦作夕阳吟。

注:叶公，指叶剑英，其诗句"老夫喜作黄昏颂，满目青山夕照明"。

# 十一

冷凳坐消春夏秋，清吟何盼贵人酬。

常橱所幸经纶涨，高壁堪欣翰墨遒。

因爱草根诗化土，缘仇棘刺笔为钩。

腾腾清气赛天地，管甚入流不入流。

注：不，可平可仄，此处平读，以避孤平。

# 十二

东风不懂尘间事，乱把雪花揉作泥。

暖起蚊蝇狂逐臭，寒消虎狼猛撕鸡。

蜘蛛趁势网罗布，猎户乘时弓弩提。

只是清污如刈草，一茬割罢更深犁。

# 十三

雪霜摇落又阳春，水碧山青万象新。

远海采油高塔竖，制空划线战鹰巡。

捕蝇捉虎不封顶，探月傲天继显身。

一任东风吹暖去，神州处处尽氤氲。

# 和寿新元老师《散吟十三章》（依韵）

# 春日郊游

中华通韵

春光明媚艳阳天，户外郊游倍爽然。

陌畔麦田腾绿浪，谷边松下涌清泉。

依依翠柳撩人面，颤颤红枝赏杏园。

醉卧斜晖欢语乐，迟迟归去忆难眠。

# 匡庐秋望

中华通韵

秋高气爽去观庐，红叶流云时有无。

仰望山崖飞瀑泻，俯观潭水落花浮。

如琴湖畔兰三径，草舍门前酒半壶。

寻觅乐天诗境在，举杯犹叹古今殊。

# 人生自吟

中华通韵

生来万事实难精，从业做人总尽情。

几所校园千点泪，数间陋室五更灯。

登山临水寻幽趣，对句吟诗觅雅风。

取舍爱求真善美，心无遗憾自身轻。

# 观《赵氏孤儿》有感

奸臣岸贾太猖狂，无视龙威尽逞强。

恶语谗言污赵氏，光天化日斩忠良。

程婴舍子儿身丧，杵臼存孤自命亡。

取义弃生民敬仰，千秋留得美名扬。

注：春秋时期晋国一场残酷的宦海之争，晋国大夫屠岸贾为了权力害死忠良赵盾，赵家赵氏孤儿除外三百多口全部被杀害。

# 汴水怡情

中华通韵

阳春三月雨蒙蒙，一叶轻舟汴水行。

陌上桃花红艳艳，河边嫩草翠莹莹。

鱼潜波底双双语，鹭舞烟汀对对鸣。

撩面清风神更爽，闲观处处总怡情。

# 棹头放歌

郊外池塘迷醉处，船移棹响荡清波。

蛙藏苇下呱呱唱，莺立莲头恰恰歌。

碧叶风摇飞玉露，琼枝雨打落红荷。

如斯美景休辜负，切任流光烂漫过。

# 闲数春秋

中华通韵

少小激情不用猜，勤修刻苦望成才。

华年动乱迷津误，岁月蹉跎志气衰。

花甲吟诗豪兴至，古稀摄影雅风来。

跋山涉水行云乐，闲数春秋诉我怀。

# 田园春景

白墙黛瓦两三家，高柳溪桥荠菜花。

娇艳红玫銜晚露，轻盈翠鸟唱朝霞。

山前小伙欢犁地，坡上姑娘笑采茶。

一幅田园诗意醉，情思缕缕叹芳华。

# 江南春韵

烟雨江南风景秀，山清水绿是谁涂。

蝶飞燕舞红花径，鱼戏蛙鸣碧草湖。

水榭楼台琴韵畅，溪桥柳巷曲声姝。

流年莫叹桑榆晚，赏画吟诗亦乐乎。

# 晚霞诗韵

中华通韵

芳年怀志苦争拼，虽遇古稀仍勤奋。

常扣键盘观世界，时翻书报看风云。

田园漫步犹为乐，诗海遨游倍感欣。

更喜夕阳无限美，彩霞万缕染黄昏。

# 诗老雁声

华发吴霜已退休，攻诗岁晚几春秋。

唐风熏染才思动，元曲萦怀雅趣遒。

缕缕闲情舒我意，桩桩心事话侬愁。

田园漫步心恬静，信笔云笺自在讴。

# 暖春入醉

和煦东风吹大地，满园春色百花奇。

翩翩蛱蝶蕊间闹，颤颤蜜蜂枝上嬉。

澹澹清波飞紫燕，依依翠柳唱黄鹂。

融融暖意留人醉，归去迟迟步未移。

# 古城新貌

古城灵璧汴河滨，洗去尘颜面貌新。

北苑草花开妩媚，南园巧石展奇神。

千年驱鬼钟馗剑，万载争雄项羽身。

无愧两淮名胜地，莺歌燕舞喜迎春。

# 赏寿新元老师书春联

迎春寿老写春联，悬腕挥毫气韵坚。

笔墨横姿龙凤舞，行云流水一酣然。
