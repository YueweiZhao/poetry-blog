# 四季曲

# 春

春雨起惊雷，春风送燕归。

春阳生万物，春野赏芳菲。

# 夏

夏风掀麦浪，夏雨打荷塘。

夏夜闻蝉噪，夏时萱草香。

# 秋

秋雁向南翔，秋霜染菊黄。

秋风吹叶落，秋月照帷床。

# 冬

冬风吹断杨，冬雪织云裳。

冬夜寒灯读，冬梅暗送香。

# 石园早春

东风疏柳吹，春水泛涟漪。

鸥鸟双双戏，离离岸草迟。

# 初春

阳气回升日渐长，风催杨柳吐芽黄。

梅腮墙外枝头艳，草露晨光鸟语香。

# 踏青

# 一

晨晓雾轻轻，闲身缓缓行。

春风杨柳岸，闻鸟唱新声。

# 二

蝶舞蜂飞春未迟，浓桃艳李斗芳姿。

堆烟杨柳宛如画，忙趁景华休误时。

# 三

户外郊游露湿衣，溪流欢唱燕双飞。

花红草绿天清朗，一抹清香新翠薇。

# 四

麦绿桃红油菜黄，佳人倩影偎花旁。

双双彩蝶翩翩舞，娇小游蜂薄翅香。

# 梨园采风

# 一

着意寻芳到下楼，梨园繁蕊素花稠。

雪香凝树气清绝，寄语春风且驻留。

注；2019.4到下楼镇梨花节采风。

# 二

朝阳古镇远名扬，几度梦中园艺场。

花海茫茫铺白锦，年年秋月果飘香。

注：2019.4朝阳镇梨花节采风。

# 三

正是一年春好时，凝香白雪压繁枝。

仙姿灵秀舒高洁，无限风情入画诗。

注：2021.4朝阳镇梨花节采风。

# 四

蒙蒙细雨晨初霁，堆雪琼英未染尘。

淡淡清香牵远客，红妆娇媚共迷人。

注：2021.4朝阳镇梨花节采风，"红妆"指赏花女子。

# 汴水情

# 一

清芬一片汴流长，十里河堤十里香。

雨润群芳开艳艳，花间裙袂笑春光。

# 二

雨霁初晴河面平，黄鹂鸣啭柳菁菁。

双双紫燕临波舞，一叶扁舟自在行。

# 三

汴河如练白云飘，堤柳青青花草摇。

双燕呢喃枝上绕，辑舟放任自逍遥。

# 钓乐

氤氲烟霭锁河津，矫手抛竿凝气神。

闲日寻幽寻钓乐，香钩提起跃金鳞。

# 渔舟唱晚

# 一

雨霁苍穹挂彩虹，落霞铺锦满河红。

渔舟唱晚画图里，神笔谁持涂抹工。

# 二

秋色苍茫秋草黄，芦花摇曳映斜阳。

波光熠熠渔舟唱，汴水流霞晚韵扬。

# 汴水晚眺

中华通韵

夕阳西下映余晖，点点渔舟唱晚归。

缕缕彩霞如火锻，轻轻烟霭盖云飞。

嘎嘎鸿雁长空去，宛转黄鹂林圃归。

夜色降临人渐退，镰钩初挂幕帘垂。

# 村野归

黛瓦红墙月季花，路旁醒目一农家。

门前溪水丝丝柳，桃李枝头染彩霞。

# 汴河晨曲

# 一

霞光缕缕日东升，雨霁初晴汴水盈。

一叶渔舟勤起早，鱼虾满载棹歌行。

# 二

青青河畔草，一夜露珠莹。

紫燕双双语，黄鹂对对鸣。

白云天际去，红日水边明。

隐隐琴声荡，悠悠香径行。

# 郊游

碧野铺金耀眼黄，蝶飞蜂舞采春香。

东坡杏蕊披朝露，西岭桃花沐夕阳。

芳草萋萋葱郁郁，烟云渺渺渺茫茫。

踏春游赏城郊外，姐妹欢声笑语扬。

# 惜春

阴雨绵绵天欲暮，徘徊柳陌惜年芳。

东风无力残英乱，几许流连几许伤。

# 荷塘

# 一

荷塘新雨后，碧叶露晶莹。

波荡采莲女，香传笑语声。

二

白云浮水两重天，锦鲤弄波摇藕莲。

翠叶似盘珠露滚，如诗如画自流连。

# 清夏

清和初夏夏花妍，红倚绿云漫若烟。

林径深深闻鸟语，欲将移步紫藤牵。

# 护城河畔

连天骤雨水流急，浊浪卷沙展势威。

岸上花丛蜂蝶杳，狂风掀起路人衣。

# 观节制闸蓄水

汴水澄澄波面阔，小舟三两画中摇。

拦河大坝云天耸，千里良畴更富饶。

（新汴河灵璧境内）

# 浍塘沟鸬鹚节

# 一

河口放鸬鹚，波花荡起时。

鱼儿波面露，渔父获珍滋。

# 二

濉河两岸草如茵，数十长竿垂吊纶。

忽见锦鳞波面跃，悠然收取乐频频。

注：2019.5浍沟鸬鹚节。

# 三

两岸花香好景时，轻舟飞棹弄鸬鹚。

鹰衔锦鲤腾波面，眼快渔人取未迟。

注：2021.5浍沟鸬鹚节。

# 熏夏闲咏

布谷声声催麦黄，平畴沃野泛金光。

村村稼穑机耕地，户户开镰粮入仓。

栀子销魂寻旧梦，兰舟放棹荡荷塘。

波移莲动惊栖露，一片哗然展翅翔。

# 雨中即景

# 一

栉风沐雨只身行，小草小花挥手迎。

路面低洼蹚水过，童年趣味悄然生。

# 二

两三蒲草水边生，剪影翩然波面轻。

细雨斜风摇翠蔓，快门伴我听蛙鸣。

# 难忘2018·8·18夜

昨夜台风温比亚，雷鸣电闪雨如麻。

行船街巷今方见，一片汪洋千万家。

注：2018.8.18灵璧遭受台风"温比亚"袭击。

# 台风过后见闻

# 一

大院居民慌，愁云锁户房。

有门难进出，搂在水中央。

注：2018.8.18台风过后一小区楼上居民生活必须品一度靠行船送至楼下，楼上居民用绳索提上。

# 二

台风见奇观，路面捉鱼欢。

地下停车处，水深三尺寒。

# 龙卷风

狂沙飞石漫飞来，房盖窗棂撕裂开。

拔起参天银杏树，乱枝碎叶满池台。

注：2008年夏，目睹了灵城遭遇龙卷风袭击的惨状。

# 台风"烟花"所感

"烟花"路径经灵璧，防患时时党干群。

肆虐狂风今事后，城乡无恙万民欣。

注：2021.7第六号台风"烟花"28日影响我县。

# 纳凉

# 一

草露霞光汴水前，徐徐陌上柳垂肩。

风吹裙袂生凉意，难得人间六月天。

# 二

紫薇簇簇柳依依，芳草萋萋露湿衣。

撩面晨风消暑气，陶然世外不思归。

# 赞夏日电路维修工

一从淋淋似水浇，高空作业上云霄。

炎炎烈日何言苦，乐得千家暑气消。

# 喜雨

阵阵轻雷催雨降，一天一水夏禾鲜。

田间喜看层层绿，农户欣迎丰稔年。

# 夏思

十里荷塘十里香，清波翠盖有蛙藏。

云愁雨骤花零落，一片飘红一片伤。

# 灵阳留梦

谁催大地换新装，沃野涵烟云影长。

白鹭翩翩讴水面，黄莺恰恰唱回廊。

雨敲荷叶声声脆，风送莲花阵阵香。

美景良辰心欲醉，归来留梦入仙乡。

# 秋怀

金风天气爽，桂子满庭芳。

熏透诗人梦，素笺留几行。

# 秋声

阵阵秋风起，萧萧桐叶飞。

层层铺满院，愁绪绕心扉。

# 秋夜

凉月挂疏桐，轻罗觉露浓。

松窗摇影动，惊梦数声蛩。

# 秋韵

# 一

幽丛霜露华，秋色染黄花。

冷艳牵骚客，吟声醉晚霞。

# 二

寻旧似陶家，东篱赏菊花。

闲吟今古句，诗酒度年华。

# 三

红叶满枝头，溪河静静流。

几行鸣雁过，诗意锁清秋。

# 蝶恋花

幽处隐孤芳，西风送晚香。

野花临蛱蝶，一任弄秋光。

# 晨韵

薄雾轻烟湖面流，白云丝柳荡悠悠。

时时禽鸟枝头唱，秋日秋光秋韵柔。

# 访菊花茶基地

菊园九月正芬芳，巧手姑娘采摘忙。

待到黄花烘焙后，一杯入口齿唇香。

注：2019.9参观本县大庙王沈菊花基地。

# 初冬

# 一

落木萧萧汴水湾，红深黄浅更斑斓。

芦花似雪临波舞，徐步景中当自闲。

# 二

# 霜侵万树最迷人，不是春花却胜春。

彩叶翩翩铺锦绣，江山如画醉良辰。

# 初雪

# 一

田野茫茫罩素装，北风呼啸玉沙扬。

千家万户农人乐，祈稔来年粮满仓。

# 二

瑞雪纷飞飞满天，冰莹剔透罩庭园。

瘦枝窗外临风立，夜半梅香更摄魂。

# 冬日

中华通韵

凤冽透寒衣，匆匆行影稀。

环城河畔柳，雪压断枝几。

# 雪中趣

皑皑白雪罩田园，村野顽童趣味牵。

小手冰凉抛碎玉，迎来飞往乐声传。

# 腊梅

冰天雪地傲严寒，独秀琼枝自在妍。

蜡染凝脂香韵意，报知春训近家前。

# 赠远

腊梅开岁寒，冷艳可人怜。

赠远芳盈手，暗香千里传。

# 元宵雪

元宵佳节雪花飘，点点星星分外娆。

飞入梅花寻不见，融融春意闹枝条。

# 小寒随感

时至小寒三九天，晨霄帘外雨潺潺。

庭园嫩草油油绿，虽至隆冬春意悬。

注：2019年家乡暖冬。

# 灵璧火车站抒怀

一声鸣咤破长空，电掣风驰跑巨龙。

盘古开天灵璧站，喜逢盛世四方通。

注；2014.12.10家乡通火车。

# 奇石文化园

# 奇石文化园

汴阳楼耸立中央，石磬流音绕画梁。

烟柳回廊相掩映，梅林幽境任徜徉。

一庭紫竹风流韵，满目红荷雨送香。

轩榭亭台多翰墨，千年古句绽华章。

# 迎春花

春风唤雨轻轻洒，津润新泥草色无。

唯见一枝黄蕊发，悄然舒展自端殊。

# 早春湖畔

伟岩湖畔立，石柳两相依。

春霭轻轻绕，黄鹂自在飞。

# 花露

繁花衔玉露，香蕊蝶蜂飞。

风过摇红影，碎珠何所归。

# 赏红梅

桃未芳菲杏未红，娇姿已放此园中。

丹唇含笑惹人醉，韵染香笺诗句工。

# 游园

乔木森森绿满园，钟灵峰秀路边沿。

画桥碧水衔诗意，信笔拈来书大千。

# 汴阳楼

波光潋滟柳丝扬，花草芬芳袖带香。

燕子楼头相对语，声声婉转诉情长。

# 九曲桥

白玉镶成九道弯，碧波倒映两桥连。

朦胧岚雾云来去，独自凭栏听管弦。

# 绘月桥

堆烟杨柳拂清风，玉砌雕栏湖面嵩。

落日余晖摇翠幕，一弯新月月桥东。

# 奇石园秋色

岸树披红伴月桥，平湖如鉴小舟摇。

縠紋微泛浮楼阁，水浣秋光分外娇。

# 秋游奇石园

金风凉气爽，石苑满秋光。

梧叶空中舞，游蜂蕊上忙。

天蓝湖水碧，露白菊花黄。

北雁鸣声早，云笺逸韵长。

# 初雪

琼花初洒洗尘埃，片片落阶情趣来。

背起相机临雅境，石园美景镜头裁。

# 雪后晨霭

冰天雪地雾朦胧，飞鸟游人踪影空。

纤柳月桥如梦幻，石园万景有无中。

# 冬日游园

# 一

风敛日高天气暖，枝头鸟雀尽哗欢。

小桥梅柳幽幽处，情洒芳园意未阑。

# 二

琼枝寒蕊醉心怀，曼妙仙姿香雪腮。

最爱南园日晴后，晶莹剔透胜瑶台。

# 寻梅

数蕊竹旁恣意开，梅园幽径自寻来。

暗香浮动撩人醉，随折一枝诗句裁。

# 隆冬见闻

# 一

枯柳衰条湖面封，舟船凝冻岸头横。

水禽无影身何处，却听顽童嬉戏声。

# 二

皑皑白雪鸟无踪，修竹青松靓逸容。

玉树琼枝斜照里，如诗如画意重重。

# 游园赏腊梅

瘦枝寒蕊傲长空，四九鲜妍紫竹东。

蜡染仙姿香韵绝，年年芳信约春风。

# 残雪

晚晴斜照里，梅蕊靓莹莹。

残雪犹然缀，清寒陶雅情。

# 雨后春山石

幽谷云雾绕，草碧柳堆烟。

雨后春山醒，一枝红杏妍。

# 醒酒石

苏子曾临张氏园，兰亭作记赋迁闲。

诗翁畅饮蓬莱醉，酒醒欣然笔墨牵。

注：灵璧张氏兰皋园一石称为"小蓬莱"苏子瞻爱之，题其上曰"东坡居士署中观此，洒然而醒"。

# 唐风宋韵石

唐风宋韵秀芳姿，画栋雕梁掩柳丝。

舞尽千年宫女怨，回眸低首亦凝思。

# 将军石

将军峰上将军立，虎气威严振四方。

铁目铮铮防进犯，太平盛世岁光长。

# 和平信使石

和平共处中华愿，友善为邻总在前。

收敛干戈消霸气，大同世界梦期圆。

# 岁月蹉跎石

风华正茂自当强，卷入潮流学业荒。

往事悠悠常感叹，蹉跎岁月误时光。

注：1966年高中毕业，高考前夕全国掀起了文化大革命，学校停课，66、67、68高、初中三届学生在校闲置了两年于1968年下放或回乡皆到农村落户。

# 虞姬园

# 虞姬园

垓下荒原新景象，亭廊楼阁著辉煌。

一园翠色连天碧，千树艳花熏鼻香。

破釜沉舟豪气在，别君饮剑挽歌长。

骚人墨客碑群立，咏叹英灵后世芳。

# 虞姬文化园感怀

# 一

帐外楚歌悲切切，乡愁缕缕惹人忧。

红颜挥剑刎香颈，血染青锋霸梦休。

# 二

策马同行征战里，风云突变霸王踌。

舍身饮剑成长别，一曲悲歌千古留。

# 三

四面楚歌起，佳人血践行。

乌江同剑刎，笑看死和生。

# 四

一堆荒冢葬芳魂，垓下当年血泪痕。

子弟八千何处去?沙场征战几人存？

# 虞美人花

袅袅悲歌垓下哀，霜鋒刎别赴泉台。

可怜血染河边草，魂化花红墓上开。

# 谒虞姬墓

佳人墓畔艳花香，独舞春光彩袂扬。

四面楚歌情故旧，魂随征战伴君王。

# 虞园广场塑像

千里楚天墟野昏，并驱齐驾结同根。

虞园合塑乌骓像，一代枭雄浩气存。

# 游虞姬园撰联

垓下楚歌起；荒原剑血流。

破釜沉舟豪气在；别君饮剑冢园留。

# 钟馗园钟馗塔观雪

# 一

朵朵雪花天女散，茫茫渺渺任风飘。

洋洋洒洒馗园驻，素裹银装分外娇。

# 二

雪霁登临环四方，城乡一派白茫茫。

凤山汴水氤灵气，瘟疫妖魔麾扫光。

# 馗峰塔

玉带悠悠环楚岫；塔楼奕奕绕秦云。

振邪除恶乾坤朗；除恶为民日月新。

利剑高悬魑魅惧；馗风横扫腐贪慌。

# 观涛阁

岸柳鸢兰烟霭里；涛阁修竹画图中。

登高远眺涛声远；扶槛俯察溪水长。

# 聆磬阁

青岫岚烟绕；素琴清韵飘。

聆磬阁幽时戛玉；钟馗园秀总流香。

# 农博园

仙境桃源映眼帘，葱茏锦绣众芳妍。

才惊叶底相思豆，又喜枝广玉兰。

冬日夏花花艳艳，春天秋果果甜甜。

涓涓流水牵清梦，如画如诗生态园。

# 农博园一隅

布谷声中夏令来，纷纷红紫竞相开。

坛花浓艳铺朱锦，洒洒洋洋满石台。
