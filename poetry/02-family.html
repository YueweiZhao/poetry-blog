# 过年（2014年）

# 团圆

思乡游子竞相还，燃竹悬联喜气传。

庭院飘香飞笑语，杯盘丰满庆团圆。

# 别离

匆匆聚散又新年，骨肉别离柴院前。

泪眼婆娑行渐远，牵儿挂女夜难眠。

# 雪阻

条条高速轿车忙，忽见空中飞雪飏。

万亩禾田皆素裹，归家迟早又何妨！

注:举家自驾返京途中遇大雪阻.

# 暗香

中华通韵

瑞雪飞扬帘半卷，深深庭院寂无言。

窗前静卧听风雨，隔案梅香香暗传。

# 过年（2015年）

# 一

思乡儿女归来早，姑嫂倾心笑语摇。

辞旧子时闻爆竹，筵开新岁闹通宵。

# 二

中华通韵

三阳开泰年来到，好运临门步步高。

春酒频频情未了，你谦我让品佳肴。

# 三

中华通韵

又闻鞭炮声，触绪更心疼。

把盏无君影，抽身泪雨零。

# 四

中华通韵

匆匆年又过，深院静悄悄。

子女纷飞去，诗书独自陶。

注：为了环保，家乡灵璧2018.12.1起灵城城区内禁止燃放鞭炮。

# 年味消（2020年春节）

新型病毒惊尘世，儿子归乡速返京。

挥手别离风雪里，柔肠牵挂已三更。

注：新型冠状病毒疫情肆虐武汉封城，儿子一家年初二速回京城。

# 除夕夜（2021年）

迎来辛丑年，二女绕身前。

子在京城远，柔肠两地牵。

# 巾帼参战本色（2020春节）

（写给工作在乡政府的小女儿）

# 一

疫情防控令如山，离院抛家未等闲。

户户排查明细细，病源阻断早回还。

# 二

小女纵然孱弱身，不分昼夜不辞辛。

同心协力妖魔阻，胜利归来洗战尘。

# 草莓吟

翠叶缀玲珑，颗颗玛瑙红。

珍馐装玉碗，尝品味无穷。

注：2020年二月防疫防控居家隔离一月余，女儿送来鲜果甚喜随吟之。

# 杏花吟

微茫微雨洗轻尘，二月田园草木新。

忽见案头芳蕊艳，女儿送我一枝春。

注：女儿折了几枝杏花插于瓶中置于案头。

# 游园遇梅开

东君先入石园来，一夜红梅满树开。

淡淡清香牵玉袂，楚腰花下映桃腮。

注：2021年正月与小女儿送我去奇石文化园拍照。

# 陪大女儿垂钓

风光旖旎水塘边，静静垂纶自逸然。

浮子下沉竿速挚，红鳞腾起落青鲢。

# 鱼筵

活蹦青鲢提上岸，炸烹熏制灶台前。

佳肴美酒餐桌满，投箸品尝滋味鲜。

# 母女仨游园

三月春风染彩霞，游园姐妹绽芳华。

丹唇含笑流丰韵，袅袅婷婷两朵花。

# 荷塘醉了谁

红荷向面开，倩影映池台。

衣袖牵香韵，蛾眉新月裁。

注：2015.8母女游湖州荻港渔庄。

# 采草莓

点点嫣红缀垄台，采莓姐妹采棚来。

纤纤玉手提篮满，二月田园梦境开。

注：2019年初春母女仨和小外孙到冯庙草莓大棚采摘草莓。

# 海底奇观

中华通韵

携孙海底乐游园，水母浑如朵朵莲。

体态轻盈时曼舞，眼前处处尽奇观。

注：2019.3带小外孙到南京海底世界游玩。

# 贺孙子高考报捷

中华通韵

金榜题名美少年，举家欢庆乐陶然。

寒窗十载展鸿志，书海一程扬巨帆。

高尚情操常历练，渊深学识再登攀。

自强不怠时时勉，振羽翱翔上昊天。

# 贺孙子录取北京航空航天大学

中华通韵

喜讯传来取北航，亲朋称赞好儿郎。

愿孙志展中华梦，科技高峰傲万邦。

# 赠孙子

及第少年欣折桂，踏枝喜鹊壁生辉。

春风得意才华展，名校园中满载归 。

注：2015年孙子赵越伟考取北京航空航天大学。

# 赠外孙

从戎投笔戍边关，英武白皙一少年。

赛马琴音催奋进，擒拿格斗尽争先。

注：2010大外孙袁思齐应征入伍，业余爱好拉二胡。

# 感怀人生

# 一

荏苒时光七十秋，如烟往事涌心头。

芳菲立志读书苦，花信成才动乱愁。

辗转讲台倾烛泪，耕耘桃李誉神州。

穿词缀句情丝染，诗海逍遥乐荡舟。

# 二

流年向晚太匆匆，七秩依然文字攻。

灯烛诗书茗作伴，芳馨染韵趣无穷。

# 三

金飞玉走鬓如霜，拙笔勤耕书丽章。

慢步柳溪花草岸，小桥流水立斜阳。

# 四

况味人生细品尝，风霜雨雪道茫茫。

闲愁淡定从容度，心似莲花一路香。

# 桑榆情

# 一

初心永放平，与世始无争。

云淡风轻去，静听花落声。

# 二

远观云海接蓝天，近赏繁花开满川。

临水临山临雅境，桑榆非晚趣依然。

# 藏头诗（作者名）

李花香韵顺时开，素雅清新莺燕来。

芹意如兰舒己性，蕙心紈质父母栽。

# 联

同仁做友；与义为邻。

幽静诗书雅；悠闲花木鲜。

轻松过一世；潇洒度平生。

为善心当乐；读书情更佳。

博爱心胸阔；无私天地宽。

处事身前想；做人背后思。

莫说他人短；毋夸自己长。

三盏两杯淡酒；千思万绪浓愁。

临水登山烟雨；粗茶淡饭诗书。

闲品诗书雅韵；怀思尘世离情。

凡事力争完美；做人总要宽仁。

恩爱数年如一梦；离怨半日成长思。

春色十分常易尽；相思半缕总难收。

孤窗孤月孤身影；空室空帷空枕头。

深院相依云聚散；空帷独卧雨飘零。

汗洒诗田陶雅趣；笔耕词野释怡情。

虚度光阴余日少；只争朝暮剩时多。

和顺一门增百福；平安二字纳千祥。

# 贺二女儿喜得贵子联

贤门今日又添麟趾；贵府他年定取功名。

# 自况联

歌佳世，缀句穿词抒雅兴；贺古稀，登山临水寄闲情。
