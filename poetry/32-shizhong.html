诗钟:一种文字游戏之作。其方法为任取意义绝不相同的两个词，或嵌字，或分咏，创作五言或七言两句，以凑合自然，对仗工整为止。相传拈题后以寸香系缕上，缀以钱，下呈盂，火焚缕断，钱落盂响，故名。

# 凤顶格[松、竹]

松涛阵阵绿林动；竹韵声声碧影摇。

# 凤顶格【春、晓】

春风吹绿千池柳；晓雨催红万岭花。

注：凤顶格，题二字，平仄不拘。两字分别嵌在上下句的第一字位置。

# 燕颔格【清、远】

风清月冷故人去；路远水长游子愁。

山清水秀天堂寨；塔远林幽拙政园。

（建园时借邻园的塔可远观）

# 燕颔格【梦、魂】

诗魂朦胧浮玉案；月魂皎洁照芸窗。

注：燕颔格，题二字，须一平一仄。分嵌在上下句的第二字位置。

# 鸢肩格【安、勤】

常思安危家庭旺；不忘勤俭事业兴。

# 鸢肩格【雄、壮】

风光壮丽天堂寨；气魄雄深紫荆城。

注：鸢肩格，题二字，平仄不拘。分嵌在上下句的第三字位置。

# 蜂腰格【雪、梅】

漫天瑞雪纷纷洒；一树寒梅静静开。

# 蜂腰格【寂、寒】

山空林寂轻烟袅；霜满星寒残月阑。

注：蜂腰格，题二字，须一平一仄。分嵌在上下两句中间，即第四字位置。

# 鹤膝格【吹，打】

屡屡春风吹细柳；萧萧秋雨打残荷。

# 鹤膝格【开、拓】

包揽诗书开视野；常挥翰墨拓襟怀。

注：鹤膝格，题二字，不可双仄。分嵌在上下句第五字位置。

# 凫胫格【乱、真】

岁岁平安休乱纪；家家和顺见真情。

# 凫胫格【云、月】

晓雾垂帘明月照；晚霞铺锦彩云飞。

注：凫胫格，题二字，须一平一仄。分嵌在上下句中第六字位置。

# 雁足格【风、味】

山清水秀天堂寨；塔远林幽拙政园。

# 雁足格【冷、香】

愠色待人容送冷；温言留客齿生香。

注：雁足格，题二字，须一平一仄。分嵌在上下句第七字位置

# 汤网格

李若梅青多澹素；芹如兰雅自芬芳。

注：寿新元先生嵌李素芹女士名。

# 汤网格【共、同、体】

追求名利休相共；体验甘辛愿与同。

注：汤网格题三字，嵌四角之三。

# 四皓格【走、马、观、花】

骑马漫游观柳绿；牧羊闲走赏花红。

走入草原观骏马；步临花海见羚羊。

醉游花海观玫瑰；梦走桃溪赏马兰。

注：马兰即马兰花

# 四皓格【桃、花、依、旧】

桃叶凋依然旧景；杏花绽已是新春。

雨打花飞香照旧；风吹桃舞韵依然。

注：四皓格，题四字，无序散嵌于上下句中，题字不连。

# 流水格【云、凝、空、翠】

云涌凝红空舞练；风吹摇翠水生烟。

# 流水格【春、归、柳、上】

春回归燕溪边柳；夏至犁牛陌上花。

注：流水格，题既要拆开不可连缀，又要不改变顺序，如溪水顺流而下。

# 云泥格【情、翠】

晨雨霏霏敲翠竹；晚晴淡淡染红荷。

柳翠花红摇晓岸；山清水秀荡晴川。

# 云泥格【峰、会】

"高峰"聚晤中南海；代表欢谈大会堂。

"金砖"互利"高峰"创；展会同赢政界推。

# 云泥格【士、超】

吟士梅亭独咏唱；忧人柳巷自超摇。

注：云泥格，题二字，一平一仄，分别嵌在上下句中第二字和第六字中。

# 睡珠格【虎、啸、龙、吟】

阵阵龙吟腾四海；声声虎啸震三关。

# 睡珠格【民、富、国、强】

大地萦回强国曲；神州描绘富民图。

党政同描强国画；干群共唱富民歌。

注：睡珠格，题四字，无顺序两两连缀，分成两组，组成两个词，两组位置对正，分嵌上下句。

# 勾股格【春、光、曲】

光阴似箭填新曲；春日如梭抒旧词。

注：勾股格，题三字，犹如直角三角形的勾和股。选其中二字，任意放在一至七对应位置，第三字嵌在隔一字以上处。

# 合咏格【山泉】

绝壁红崖松郁郁；巉岩翠谷水潺潺。

# 合咏格【雪】

越墙进院千家玉；盖地铺天万树银。

# 合咏格

（禁用：血汗冰霜冬松竹傲八字）

凛冽朔风独蕊艳；浅清疏影一支香。

注：合咏格，取一词为题，两句共同描述题面，忌犯题字，忌用所咏对象的别名。

# 分咏格【水、书籍】

诗文吐韵篇篇雅；花露流芳处处香。

# 分咏格【羊、月亮】

辽阔草原飘白锦；无垠天幕挂银钩。

注：分咏格，取两个词语作题面，上下句各吟一题，可顺可倒。句中用字不能犯题，也不可用所咏对象别名。

# 嵌咏格【咏竹嵌"紫"字】

斑痕点点湘妃泪；紫节竿竿舜帝情。

注：嵌咏格，一题一联，禁犯题字，必须嵌入一个与题无关的字，嵌位无束博。

# 魁斗格【怀、请】

怀善抱清留玉洁；修身养德著冰清。

# 魁斗格【长、征】

征雁排排穿舞过；天鹅只只逐云长。

注：魁斗格，题二字，不可双仄，分别嵌于上句首字和下句尾字位置。

# 小鼎峙格【中、华、龙】

长白山中彪虎啸；琼华岛上巨龙腾。

（琼华岛在北京北海公园内）

注；小鼎峙格，题字必须三平或三仄，三字所嵌位置，一句占中，另一句占第二字和第六字处。

# 三星格【稻、花、香】

稻穗香飘云岭外；荷花馨洒水池旁。

注：三星格，题三字，所嵌位置是上下句中任何不隔位的三字处，连成一个或正或反的等边三角形。

# 蝉联格【树明】

斜风细雨穿千树；明月清露洒万家。

注：蝉联格，题二字，不可双平。分别嵌入上句尾和下句首处。

# 卷帘格【翠、湖】

晚霞流彩红湖荡；初日浮岚翠幕垂。

注：题二字，可平可仄，上句所嵌之字须低于下句所嵌之字一位。

# 太极格【风、月】（三唱）

雾散风馨清夏始；霜临月洁冷秋深。

注；太极格，题二字，须一平一仄

# 笼纱体【山海】（用隐或暗）

连天波底游鱼乐；拔地峰巅飞鸟愁。

千层碧水千层浪；一片孤城万仞山。

注：笼纱体，题二字，字不明嵌，而是暗嵌。

# 晦明体【红、花】

红笺烛泪闺中怨；玉字锦文笔下生。

雨霁晴空浮瑞彩；风驰芳园扫残雪。

注：晦明体，题二字，一字暗嵌，一字明嵌，位置不限，明暗不限。

# 六逸格【秋、日、菊、花、傲、霜】

菊黄蕊冷霜天傲；桂馥花馨秋日芳。

注；六逸格，题六字，散嵌无序，只许一处两字相连，余字不得连缀。

# 竹节格【观、咏、山、松、水、竹】

秋观溪水云山景；冬咏雪竹梅竹情。

注：竹节格，题六字，不论顺序，两两对正，嵌于上下句中二、四、六位置。

# 八龙格【三秋桂子、十里荷花】

三秋桂岭枝盈子；十里荷塘波映花。

注：八龙格，题八字，无序散嵌，题面可二字相连，最多三处。三字相连，只能一处。连字不超三处。

# 居易格【陶渊明、王炎】

门前五柳藏春燕；岭下双溪噪夏蛙。

（陶渊明别号五柳先生，宋词人王炎号双溪）

注：居易格，上下两句整嵌入一位名人的名或字或号。
