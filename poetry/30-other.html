# 端阳节感怀

# 一

中华通韵

悠悠岁月长，千古汨罗江。

屈子冤魂葬，离骚吟断肠。

# 二

端阳祭屈原，竞渡赛龙船。

江水悠悠痛，棹歌思楚贤。

# 守望

良人创业他乡走，少妇持家小院留。

侍奉高堂晨与暮，伴陪幼子夏和秋。

长天短夜薄身累，孤枕寒衾独泪流。

趁意月光窗外照，一霄幽梦到琼州。

# 夜

中华通韵

苍天碧海转瑶盘，谁晓姮娥夜夜寒。

应悔月宫孤寂苦，那如恩爱暖人间。

# 悟

晨曦初照静，万物悄无声。

幽境红香满，蓦然禅意生。

# 闺怨（观图有感）

# 一

中华通韵

杜宇声声不忍闻，春风难解女儿心。

无端旧梦萦怀绕，雨打梨花深闭门。

# 二

春去夏来愁日长，闺中无趣耗时光。

蛾眉微蹙门扉立，湘水望穿不见郎。

# 三．送别

波面茫茫舟渐远，梨花带雨泪纷纷。

佳人江畔迟迟立，期盼云笺传雁文。

# 四．思远

中华通韵

晓光残梦破，窗外雨濛濛。

江畔孑然立，渺无舟棹行。

# 五．雨霁江山晚景

远山似黛万千层，一叶扁舟画里行。

翠竹斜阳留晚照，轻烟薄雾雨初晴。

# 至立夏

# 一

燕子呢喃穿柳荫，黄瓜豌豆味清心。

榴花欲放樱桃熟，万物生辉初夏临。

# 二

蚕娘素手采桑忙，小麦杨花正灌浆。

绿叶丛中青杏缀，田园四月好风光。

# 白露节气有感

蓝天鸣雁高，芦苇任风摇。

晨露莹莹白，秋江秋意娆。

# 悲秋

一阵秋风一阵凉，一川烟草一川黄。

一声孤雁一声远，一片飘零一片伤。

# 寒露节气有感

中华通韵

竹外黄菊玉露琼，秋风萧瑟落叶红。

长空仰望南飞雁，捎去相思日日浓。

# 赞抗疫一线英雄

疫情突发真情在，医圣悬壶何惧难？

勇士驱瘟奔一线，坚心必保国家安。

注：2020年春节武汉突发新型冠状病毒疫情，全国各地抽调医务工作者奔赴武汉救治感染者。

# 致敬民族脊梁——钟南山院士

疫情突发祸江城，院士八旬身起征。

沥胆披肝除瘴戾，狂澜力挽楚天晴。

# 共克时艰

深闭庭门锁户中，万城小巷大街空。

全民防疫阴云散，万里神州旭日红。

# 风雨无阻

风雨飘摇黄鹤楼，瘟神肆虐满城忧。

万千医护援三镇，疫患铲除安九州。

# 风雨岁月

书香茶暖锁闲庭，心系江城常划屏。

酣战瘟神抛热血，感时恨疫泪零零。

注：时时从手机中查阅新闻，了解武汉战疫情况。

# 疫情防控月余始出门

# 一

隋堤春水前，忽见腊梅妍。

馥郁依如旧，身心顿释然。

注；因住在公路边，宅院没封闭，稍自由点，当疫情稍退时迈入大自然，心中的快慰无以言表。

# 二

疫情渐退出庭门，众鸟和鸣绕杏村。

淡淡素华枝上少，昨宵风雨殒芳魂。

# 三

户外郊游露湿衣，溪流欢唱燕双飞。

花红草绿天清朗，一抹春阳新翠微。

# 观图有感

昨宵风雨细无声，湿润朱桥满落英。

凌乱丛丛怜此际，香魂流韵溢诗情。

# 岁除随记

中华通韵

霜尽和风吻，欣然野水前。

桥头梅蕊闹，枝上柳芽跹。

守岁歌国泰，端杯庆瑞年。

钟声敲夜响，华夏换新天。

注：2020年起始新冠病毒疫情席卷我国大江南北，一年来在中国共产党及全国人民的不懈努力下，我国新冠病毒得到有效控制。2021年过了个祥和的春节。

# 捣练子·晨游汴水畔

静悄悄，步轻轻，岸柳青青枝绕莺。 堤下麦苗翻绿浪，陌边花草织芳丛。

# 采桑子·龙湖晨曲

清波短棹龙湖水，薄雾轻烟。彩霞斑斓，何处笙歌缠意绵。 鹭鸥掠起飞南岸，一片哗然。声送芳鲜，难忘人间四月天。

# 一剪梅

曾记同游植物园，花径蜿蜒。清波一片映蓝天，荷叶圆圆，珠露圆圆。 明媚春光好逸然，莺燕歌欢，蜂蝶舞欢。闲聊往事笑依栏，细语绵绵，情意绵绵。

# 清平乐

冰轮如昼，丹桂飘香透。兴至披衣村外走，草露闲牵衣袖。 碧空如洗无边，朦胧黛色山峦，鸿雁一声掠过，心扉飞上云天。

# 减字木兰·汴河晨韵

平明河畔，绿树红花双飞燕。盈袖香风，颤颤枝条晨露莹。 汴流似练，云影悠悠摇水面。一叶兰舟，欸乃歌声诗韵柔。

# 天仙子·少女忆

拂面春风盈袖香，微汗秋千解罗裳。欢声笑语洒潇湘，眉颜细，鬓云扬，秀靥粉颜映夕阳。

注：怀念少女时期就读的灵师附小，体育设施一应俱全，最难以忘怀的荡秋千的快乐。

# 浪淘沙·惊梦江南

烟柳舞翩跹，芳草连天。斜风细雨濯青莲，罗伞红裙深巷远，醉目江南。 惊雀扣幽帘，思绪潺潺，字里行间，剪一段春光静美，温润心田。

# 鹊桥仙·早春

青峰扫黛，白云飘絮，隐隐犁牛散牧。小桥流水绕孤村，棹歌起，惊飞栖鹭。 津东修竹，津西花树，香气随风轻吐。杏桃初绽笑含羞，踏春早，韶光恐负。

# 巫山一段云·咏梅

忽见琼苞破，寒梅露靓装。一枝独秀小园旁，玉蕊吐幽香。 知报春来早，闲庭倚绮窗。娇颜盈目醉心乡，疏影映

斜阳。

# 浪淘沙·游太行万仙山

游览万仙山，顺势蜿蜒，行车飞速洞中穿。绝壁红崖峡谷险，触目心悬。 河汉落瑶川，瀑响泉喧。层峦叠嶂亘无边，裸露层岩红欲染，天下奇观。

# 定风波·登庐山

一路登行石径间，清芬扑面接蓝天。烂漫山花香引路，高处，蓦然踏上白云边。 俯瞰绿荫遮嶂远，危险，仰观飞瀑落幽潭。不觉斜阳西照暮，归去，晚霞牵梦梦难眠。

# 巫山一段云

远岫青如黛，清溪静静流，白云浮水荡悠悠，平野画屏幽。 日暖花争艳，鹂莺恰恰讴。丝丝烟柳弄轻柔，春色锁神州。

# 长相思·游沈园

雨濛濛，雾濛濛。桥下清波堤上红，故园寻旧踪。 薄欢情，薄世情。壁上依稀词义浓，伤怀怨九重。

# 钗头凤

波光潋，春花艳。沈园文迹残墙嵌。池台柳，依如旧。鸳鸯佳偶，休书泣谴。怨，怨，怨! 思初恋，愁无限。玉消香殒芳魂断。云僝僽，眉峰皱。空留游苑，叹惊鸿远。念，念，念！

# 浣溪沙·夏日滨河公园

# 一

汴水盈盈两岸平，隋堤柳翠草青青。燕飞鱼跃有蛙声。 簇叶繁花花滴露，斜风细雨雨初晴。时闻布谷树头鸣。

# 二

十里隋堤垂绿杨，一湾汴水逐波长。渔舟放棹网鱼忙。

远眺凤山闲翠色，近观蜂蝶采花香。游春姐妹笑语扬。

# 探春令·惜春

融融暖意，醉人天气，单衣试换。李花淡淡桃花艳。惠风畅，香飘远。 芳郊晓踏溪水岸。柳烟如画卷。紫燕斜，缓缓舟船，春且住下堪留恋。

# 柳梢青·初夏

烟柳轻盈，莺歌燕舞，绿野平畴。细雨初休，远山似黛，云霭悠悠。 徐徐紫陌闲游，放眼处，兰溪碧流，曲折蜿蜒，谁家女子？波荡莲舟。

# 好事近·晚秋

芦苇任风摇，秋瑟瑟，秋光老。霜染枫林红遍，看江山妖娆。 炊烟袅袅渐黄昏，青霄雁声杳。凉月夜轩窗下，听琴音殊妙。
