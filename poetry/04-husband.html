# 素芹姨晚年攻诗有感

诗友王新赠

中华通韵

庭深院亦深，闹市隐柴门。

虚栅掩闲客，精灵迎故人。

茶舒一口井，书遣四时云。

才子仙游后，终年作苦吟。

注；"精灵"即家养小宠物。

# 寄亡君

# 一

昔日同林鸟，营巢共此生。

相怜相比翼，何故却先行？

注；夫妻白手起家，节俭度日，直到2019年我们五十多岁时才建成属于自己的两层小楼。

# 二

清晨来到长眠处，尚未开言泪已流。

摆上果珍斟满酒，悲天恸地痛伤俦。

# 清明

# 一

携儿带女凤凰山，香烛鲜花呈墓前。

忆昔千般情与爱，念夫思父泣声连。

注；"凤凰山"指凤凰山公墓，2011年丈夫去世的第一个清明节。

# 二

阵阵凄风暮野沉，青松岗上断肠人。

如烟往事心头涌，又是清明泪雨频。

# 三

琼花漫漫气氤氲，芳溢馨香蜂蝶勤。

惆怅东风难识意，一支香雪祭亡君。

注："香雪"即梨花。

# 祭亡夫三周年

阵阵松涛天地昏，凤凰山上祭亡魂。

坟前苦酒三杯满，墓畔哀歌四野存。

冰雪寒冬谁伴暖？凄风冷雨梦同温。

纸钱化羽纷飞去，泪湿衣襟缕缕痕。

# 重阳登凤凰山

# 一

霜菊老重阳，梦牵登凤凰。

云銜山野白，草伴柳枝黄。

寒冢怜风冷，新碑泣露凉。

潇潇珠泪雨，滴滴湿仙乡。

注：丈夫生前重阳节，曾同从凤凰山南麓登至顶峰。

二

今岁又重阳，何由登凤凰？

秋风吹旧梦，霜叶染孤房。

昔日双携手，斯时独断肠。

黄昏啼雁远，唯有泪千行。

注：2014年重阳节再次独自登凤凰山。

三

凄风冷雨透心凉，遥望家乡西北方。

难忘昔年重九日，凤凰山顶话朝阳。

注：2019年重阳节风雨潇潇。

# 梦

# 一

秋爽晚风凉，梧桐落叶黄。

何时圆己梦，月下伴君郎。

# 二

寒蝉惊梦醒，秋夜月融融。

君似床前立，欲牵唯恐空

# 三

惺忪睡眼开，君似梦中来。

清秀容颜瘦，不知垂泪腮。

# 四

案上堆书卷，厅堂垂吊兰。

窗前明月共，梦里一时欢。

# 五

荷藕满池塘，微风花送香。

岸边斜倚语，梦断泪沾裳。

# 六

飞花化羽漫天杨，梦入桃溪仙路长。

烟雨满山何处是，望穿泪眼两茫茫。

# 七

劈柴烧火灶前忙，欣喜夫君回故乡。

丰盛佳肴杯酒满，西窗明月照帏床。

注：梦境，七十年代在外地工作的丈夫回家来。

# 八

惊喜与君逢异地，中山装整笑相迎。

容颜孤俊温柔语，梦断更阑痛失声。

注：梦境依然是七十年代的模样。

# 春望

碧野铺金耀眼黄，汴流如练柳丝长。

风催月季株株艳，雨润蔷薇朵朵香。

对对鸳鸯游水面，双双紫燕绕房梁。

炊烟袅袅斜阳暮，缕缕愁思向凤凰。

# 春愁

柳絮穿庭户，小楼身影孤。

何时风雨至？濯尽我繁芜。

# 春思

# 一

独步汴河香陌湾，汴流如练散轻烟。

频频回首销魂处，细雨霏霏思渺然。

注：丈夫在世时，二人常散步于汴河堤。

# 二

乍暖还寒冷雨凄，淅淅沥沥夜阑时。

浓愁寂寂难成梦，起伴烛花收泪诗。

# 三

春思万缕绕心头，不解愁眉泪暗流。

斗转星移天欲晓，西窗夜幕挂镰钩 。

# 夏思

# 一

春光老去百花残，红雨纷纷不忍观。

切莫凭栏思绪染，幽情难寄夜阑珊。

# 二

草陌茵茵鸟雀啾，绵绵丝柳织成愁。

红尘牵手随风逝，泪眼婆娑空自留。

# 三

夏花不语草无声，阡陌纵横缓缓行。

觅觅寻寻空有泪，之言片纸寄离情。

# 四

昨夜小楼风雨声，孤灯相伴到天明。

闲翻书本皆无趣，心事那般难理清。

# 秋思

# 一

独自栏杆倚，清辉漫野霜。

夜阑无睡意，银汉两茫茫。

# 二

秋深丹桂落，霜降菊花黄。

征雁鸣声渺，凝眸望断肠。

# 三

月冷霜清征雁飞，萧萧落叶已成堆。

栏杆独倚楼头望，无尽相思谁诉陪！

# 悲秋

# 一

溪边见草黄，心里顿生凉。

树上寒蝉泣，秋声诉断肠。

# 二

秋雨绵绵霜雁杳，烟云十里锁寒桥。

小窗独揽凄凉意，清冷闲庭黄叶飘。

# 三

瑟瑟西风摇落叶，茫茫旷野草花衰。

诗书寂寂吟凉夜，窗下声声促织悲。

# 四

魂飞梦断两茫茫，耳畔秋声万里长。

金井梧桐飘落叶，残荷衰草染冰霜。

一行鸣雁高天去，几点昏鸦枯树藏。

愁绪满怀銜烛泪，诗书作伴细思量。

# 望怀

# 一

幽兰深院栽，垂蕊悄然开。

不与群芳艳，留香待月来。

注：以兰自喻,思念亡夫，期盼夜里梦见。

# 二

又见杏花开，忆君同植栽。

欢欣情境在，泪眼自徘徊。

# 三

秋风万里散尘霾，鸣雁长空一字排。

捎去相思松作笔，云笺涌泪湿天阶。

# 四

一宵落叶满河池，旷野无声细柳垂。

不见长空南去雁，书笺难付锁愁眉。

# 夜思

# 一

月光淡淡洒孤窗，夜色沉沉似雪凉。

愁绪盈怀思切切，人间天上两茫茫。

# 二

滴答床前钟摆声，心乡惊碎梦难成。

西窗夜语留空忆，思念陪侬又五更。

# 三

寂夜何由泪泗横？断肠声里忆平生。

思量往事如遥梦，斜月西窗月半明。

# 四

飒飒西风落叶声，遥遥凉夜月光清。

凄然涌起旧时景，唯愿逢君梦里迎。

# 五

月光清冷洒窗棂，卧听庭前寒蛩声。

又是一霄难入梦，思思回忆到天明。

# 七夕随感

年年七夕鹊桥通，织女牛郎幽会中。

多少离人多少痛，红尘难觅泪朦胧。

# 京城散步随感

日日只身行，悲从心底生。

京城曾执手，往事总牵萦。

注：退休后我们曾一度在北京接送小孙子上学，2011年后独一人了。

# 素笺

素笺淡墨已苍黄，五十余年真爱藏。

心语行行催泪下，又萦初遇柳溪旁。

注：翻阅恋时书信的往来。

# 缘结禅心

碧野铺金耀眼黄，登高望远莽苍苍。

红墙隐隐白云处，花径深深寺院旁。

善女信男来又去，晨钟暮鼓落还扬。

心生禅意佛前拜，来世情缘愿久长。

# 十载吟

（2021年夫君仙逝十周年）

# 一

春去又春回，十年孤枕思。

耽情吟寂夜，深院影相陪。

# 二

十载空帷心宛昨，思量往事泪纷纷。

孤灯寒夜诗书伴，笔下寄情常忆君。

# 三

悠悠十载鬓添霜，追忆与君登凤凰。

佳节重阳今又度，云笺蘸泪咏离殇。

# 四

思念多凄苦，双眸热泪盈。

临窗听雨落，推户接风轻。

烛案敲诗句，书笺留绮情。

闲行寻雅意，乐此度余生。

# 五

离别红尘整十年，鲜花三束醒坟前。

灵香燃点祈心愿，苦酒满斟温冷泉。

句句呼君君影杳，声声哭泣泣声传。

朔风凛凛松涛起，雀噪乌啼彻墓田。

# 江城子·思君

孤房弹泪思悠悠，忆飞鸥，叹淹留。手捧衾衫，寂寂复添愁。夜静更深人不寐，新月落，下西楼。

# 如梦令·梦君

昨夜与君相拥，枕畔铃响惊梦。启目四周寻，凤竹西窗摇动。痴梦，痴梦，月暗风寒霜重。

# 南歌子·思

百草棵棵衰，心笺字字愁。井梧零乱又知秋。一任清寒漠漠上西楼。

# 南歌子·思

雨润苍山翠，风摇桂蕊香。一声鸣雁寄南方。万缕愁思字字诉衷肠。

# 采桑子·新岁祭亡夫

倏然仙逝人憔悴，夜夜无眠，日日无言，泪眼滂沱半掩帘。 雪飞窗案知新岁，儿女团圆。老少团圆，同敬三杯到九泉。

# 采桑子·春愁

诗书几案残灯伴，字里行间。心事笺笺，遥想九冥眠未眠。 花前月下良宵逝，苦不堪言。红泪潸潸，冷雨凄风又一年。

# 采桑子·春愁

春风吹醒销魂梦，芳草萋萋。那是何时，携手栏杆相互依。 良辰美景成追忆，春燕銜泥。枝上莺啼，寂寞空楼孤枕欹。

# 捣练子

星轨隐，月光清，深院空房犹忆卿。心语万千和泪叹，诗书作伴到天明。

# 捣练子

禅院静，寺房清，袅袅梵音绕鼓声。过眼烟云皆散尽，青灯摇曳伴余生。

注：丈夫辞世后，一度曾心生此念。

# 天仙子

（单调）

烟雨微茫细细香，罗伞轻称偎君旁。两情相约远山长，难相忘，驻心乡，思绪蔓延锁碧窗。

# 天仙子

金井堆堆落叶愁，一声啼雁去晚秋。沉吟往事锁眉头，容憔悴，泪清流，天上人间两处忧。

# 天仙子

寂寞庭前移月影，小院深深独守静。西窗夜语早成空，愁难省，人时病，只待今宵留好梦。

# 望江东

双燕呢喃绕烟柳。忆往事，空回首。鱼传尺素两相守。伴左右，心弦扣。 深深小院亦如旧。寂寂夜，思良久。布衾孤枕已凉透。又风雨，人僝僽。

# 巫山一段云·春愁

河畔呢喃燕，兰香曲径幽。春花惹目忍回眸，烟柳总牵留。 曾是同携手，今朝觅旧游。伤心愁绪满萍州，泪雨洒孤舟。

# 巫山一段云·秋思

落木纷纷坠，芳消蜂蝶忧。几声南雁去无留。又是一深秋。 往事空回首，年年岁岁愁。挑灯夜语早停休，寂寞伴空楼。

# 点绛唇·秋思

又是秋来，卧听窗外潇潇雨。声声如诉，历历相思苦。 夜半衾凉，心绪孤难付。轻梦语，一时小叙，雨打风吹去。

# 玉楼春

红笺翻阅如初见，逝去韶华重又现。那年那月那一天，相会村旁溪水畔。 两情相悦许心愿，比翼双飞生死恋。相濡以沫花甲年，天地茫茫思涕泫。

# 鹧鸪天·重阳

九九凭栏楼上边，秋高气爽艳阳天。桂香菊放竹兰韵，美景良辰花甲年。 思切切，忆从前。重阳携手凤凰山。登高望远君知否？今又重阳独自怜。

# 虞美人

夜阑漏断情难了，风雨横窗扫。萧萧落木惹人愁，往事桩桩萦绕在心头。 记得初见羞赧语，盟定双飞羽。两心相悦结良缘，世事无常只影泪潸潸。

# 虞美人

波光云影交相好，莺雀枝头绕。京城漫步玉渊潭，烟柳画桥轻语倚栏杆。 故园依旧人非昨，触景愁眉锁。何方隐隐子规啼，细雨霏霏心绪更凄迷。

# 秦楼月·清明

多少事，千丝万缕心头起。心头起，青梅竹马，雁书频寄。 春风阆苑结连理，数旬恩爱成空忆。成空忆，九泉孤寂，问君何倚？

# 秦楼月·清明

凭栏看，小楼只影空怀远。空怀远，年年柳色，探春孤览。 今时心月缺难满，凄风冷雨愁肠断。愁肠断，清明时节，问声寒暖。

# 木兰花

紫菊疏，黄叶满。飒飒秋风临小院。摇竹影，韵萧然。慵倚槛门愁绪染。 望长空，南去雁。捎寄云笺心语蔓。思切切，意阑珊，寂夜悠悠书作伴。

# 青玉案

只身孤影回廊处，暗香透，蔷薇露。梁上双双燕子语，花上翩翩蛱蝶舞。谁知侬苦？依然如故。美景全辜负。 时光荏苒流年去，春夏秋冬四时误。万念皆空情已郁。百无聊赖，转身入户。寂寞听阶雨。

# 行香子·惊梦

满月盈盈，溪草蛙鸣。手牵手步履轻轻。遥遥青岫，渐渐胧明。叹星儿暗，月儿隐，露儿莹。 依依话别，难舍离情。历经年故地重逢。双双泪眼，幽梦偏惊。恨枕头凉，心头雨，枝头风。

# 玉楼春·秋怀

秋风小院黄花地，蕊冷香寒蝶不戏。昔年牵手喜登临，今日依栏独自忆。 曩时与尔缘如蜜，鸿雁频传无限意。倏然银汉痛悲离，期许来生仍比翼。

# 踏莎行·春愁

朝露沾衣，晨光散雾。一轮晓月枝梢处。莺飞草长杏桃红，芳郊阡陌独徐步。 兴尽人慵，云收日暮。诗书凉枕皆无趣。万千思绪夜阑珊，空帷欲语和谁诉。

# 小重山·秋思

树上寒蝉鸣不休。频风烟草乱，已深秋。庭园寥落怅忧忧。花槿老，残菊抱枝羞。 漠漠上层楼。霞收新月近，倍添愁。当年同照也如钩。盈盈泪，思绪涌心头。

# 蝶恋花

豆蔻年华萍水遇。鸿雁传书，一片芳心许。凤侣红尘朝与暮。春花秋月良辰度。 宿命难违留不住。泪眼愁肠，残照穿庭户。梦里依稀回老屋，掩帘欲诉啼蝉误。

# 唐多令·忆

堤柳絮如烟。田园春水前。燕呢喃，舒展眉眼。笑语盈盈芳草地，柔风动，扣心弦。 窗外月光寒。清辉洒碧帘。寂无眠，潮涌波澜。起坐沉吟书几案，幽梦远，雁难传。

# 少年游·秋怀

菊黄露白已清秋。往事怅悠悠。西窗细语，霜天遥夜，新月柳梢头。 沙洲残照双栖鹭，兀自又添愁。落木萧萧，蛩声如诉，野渡泊孤舟。

# 菩萨蛮·漫步月坛公园

森森乔木中天秀，幽幽小径兰香透。菡萏立蜻蜓，柳拂邀月亭。 闲心同漫步，好景没辜负。暮色渐临园，一轮明月悬。

注：儿子居住月坛公园旁边，在京时几乎每天都要去那里散步。

# 柳梢青

# 一

凉夜星稠，婆娑桂影，庭院清幽。独上层楼，遥遥天际，幕挂镰钩。 蓦然敛起眉头，忆往事，潸然泪流。月下何年？与君初见，欲语还羞。

# 二

独上西楼，回眸窗外，新月如钩。夜色茫茫，四周寂寂，思绪悠悠。 与君别柳溪头，绿波荡，惊飞鸥鹭。岸草青青，繁花簇簇，一叶行舟。

# 醉花阴

村外荷塘幽僻处，衣袂沾芳露。旷野寂无声，偶有蛩鸣，凉月中天幕。 期期相会相携手，相觑相倾诉。桂影渐移东，朗夜清风，君共良辰度。

注：与君相恋于1968年的秋季。
