# 忆同窗

春秋五十忆同窗，昔日那排红瓦房。

逐梦追舟灯下读，清汤薄粥亦留香。

注：六十年代皆住校就读中学，国家经济匮乏，生活贫困。

# 忆韶华

就学灵中六署寒，逐舟追梦志如磐。

挑灯夜读无休日，求索新知何畏难。

# 同窗聚

同窗心绪不难猜，往事迢迢寄梦台。

疏鬓斑斑灵璧聚，离情别绪泪盈腮。

注：2015.10我们高（三）2班同学相聚于灵城。

# 重阳寄同窗

云笺雁字长，篱菊正芬芳。

又是重阳日，天涯共举觞。

# 新春寄同窗

荏苒时光又一年，举杯千里意绵绵。

同窗泪眼几多事，海角天涯共怅然。

注：2017年春节于京城。

# 师生情

金风送爽喜相迎，畅叙师生别后情。

同敬三杯呈谢意，声声心语道康平。

注：2015.10在凤山酒楼我们班部分学生与老师团聚。

# 师生京城聚

师生晚岁聚京城，斛影杯光离别情。

追忆校园多少事，欢颜笑语吐心声。

注：2019春节在京的老三届学生宴请焦玉英老师。

# 老三届联谊会有感

# 一

桂子飘香寻故踪，校园非复旧时容。

少年绮梦宛如昨，白首相逢情更浓。

# 二

五十春秋返校园，师生懽聚尽开颜。

同台演出展才艺，曼舞清歌曲韵湲。

# 和同学"桂花又开"（依韵）

微信条条频往来，重温往事逸情怀。

恰逢桂子馨香送，追忆青春思绪开。

# 流金岁月

（老三届联谊会朗读篇）

求学灵中难忘却，同窗六载友情长。

常思教室残灯读，更念草坪笑语扬。

三尺讲坛传道业，满园桃李正芬芳。

才华正茂豪情起，意气方遒斗志昂。

漫步校园明月夜，畅谈理想柳荫旁。

风云变幻时局荡，岁月蹉跎功课荒。

收拾行囊回故里，听从号召去他乡。

机关部队留身影，桌案倦容映烛光。

工厂田头挥汗水，苍颜鬓发染吴霜。 风风雨雨齐心力，坎坎坷坷挑大梁。

不负师恩呈硕果，发挥才智著篇章。

为民为国青春献，克俭克勤终日忙。

细味人生无悔怨，喜迎盛世喜荣昌。

桑榆闲适频寻梦，书海遨游再起航。

歌赋楹联舒雅兴，唐风宋韵染诗行。

听泉踏浪临南国，赏雪聆松赴北疆。

锦绣河山收眼底，文明华夏屹东方。

春秋半世故园聚，浊酒一杯丹桂香。

潇朗容颜寻旧影，迷离陈梦涌心房。

沧桑历尽老三届，珍爱余生安乐康。

注：2018.10.5在母校灵璧中学举行老三届师生联谊会，数百名师生从祖国的四面八方聚拢来。

# 母校萦迁随感

汴水灵山氤紫气，萦迁新校育贤良。

名师济济书声琅，修竹苍苍桃李芳。

正学遗风传万代，精英神笔谱华章。

蟾宫折桂频捷报，立德树人名远扬。

注：2020.5母校灵璧中学迁址于灵城北郊。

# 忆少年·老三届联谊会

金秋时节，金风送爽 ，灵中相聚。真情话别绪，洒校园欢语。 母校登台重拾取，意融融，曲赋歌舞，展风采依旧，看夕阳鹤翥。
