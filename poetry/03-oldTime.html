# 家燕

# 一

雨霁苍山翠，满园桃李芳。

呢喃小燕子，檐下去来忙。

# 二

门楣留燕路，剪影翩然顾。

梁上旧巢环，欢声朝与暮。

注：主房的二檩条上燕子筑有巢，父亲做有燕盘挂在檩条上接燕子的粪便 ，又把门环加长点，锁门后，燕子能自由进出。

# 儿时家园

# 一

河畔几间茅草房，东墙修竹石榴行。

西窗月季蔷薇艳，春燕銜来小院香。

# 二

烟霞流水绕孤村，花放柴门满目新。

双燕穿堂擦耳过，羔羊鸡子唱明晨。

# 三

春风春雨涨春池，紫燕婉转穿柳枝。

村野村烟轻袅袅，浣衣笑语荡涟漪。

# 四

炎炎夏夜露无声，静躺凉床宅院庭。

时见飞星河汉渡，芳丛闪闪掠流萤。

# 难忘田园二月天

中华通韵

轩窗香韵入，帘外杏桃妍。

芳蕊游蜂闹，庭园泥燕穿。

蛰雷惊大地，细雨润青川。

整地南瓜种，提篮荠菜剜。

椿芽枝上采，柴草灶头添。

新酒杯中绿，美食盘里鲜。

蓬门飞笑语，把盏祝春天。

注：儿时家，灵城西南角城外，主房三间、厨房一间皆草房，大门前是灵宿公路，路边溪水清澈流往环城河，鱼虾可见。西北方眺望凤凰山清晰苍翠。父亲生活富有品位，临窗的花园里植有很多花草，如月季、蔷薇、何首乌等。院子里外栽植有枣、杏、桃、石榴等果树及其它香椿、花椒、槐等树木。

# 儿时趣

落霞晚照彩云飞，缓缓犁牛村野归。

丰稔年头欣稼樯，稚童嬉戏麦场围。

# 春日

常绕麦田寻荠菜，满篮嫩绿任由裁。

斜阳晚照炊烟起，一抹清香扑鼻来。

# 小小村庄

炎炎夏日槐荫下，邻里笑谈三两家。

午饭南瓜汤碗捧，石台围拢话桑麻。

注："桑麻"即农事。小村庄仅三户人家，家门前两颗大槐树下有两块方方正正的青石块，亦桌、亦凳。六、七十年代，夏季自家园地的南瓜熟了即是主食，中午各自端着饭碗到树下乘凉、聊天、谈农事，其乐融融。

# 武陵春·儿时家园

春燕銜泥穿小院，细雨洒花前。月季蔷薇窗外边，馥郁满庭园。 缓缓溪流涨两岸，轻棹荡青莲。紫陌欢歌笑语传，青岫远，淡云烟。

# 思母

中华通韵

慈母手中针线穿，为儿灯夜做衣寒。

承欢膝下俄惊醒，寂寞孤轮照枕边。

# 梦端阳

艾草插门旁，厨房苇叶香。

唤儿惊梦醒，慈母在何方？

注：儿时农家贫苦，端午节采几片苇叶和糯米一起煮粥，堪称美味了。

# 念父

啪啦劈呖珠算忙，随父前来在账房。

伏案持筹精细细，今宵梦里小时光。

注：早年方圆数十里疑难账目请父亲清算，曾在县城百人珠算比赛中名列前茅。

# 2015中元节

中元节到倍添伤，父母坟前泪两行。

日下双亲眠岭上，梦间儿女绕身旁。

几盘果品难行孝，数叠纸钱焚断肠。

永记家风传代代，走端坐正慰高堂。

# 母亲节（2019）

绵绵细雨洗埃尘，漫步田园花木新。

思念顿时和泪采，夜来梦里送双亲。

注：母亲节漫步在田园，芳草萋萋馨香四溢，随采野花一束欲送父母。

# 清明祭双亲（2020年）

中华通韵

清明祭扫泪纷纷，衰草深深掩墓坟。

苦酒三杯难尽孝，鲜花一束仅随心。

松涛阵阵同悲泣，杜宇声声不忍闻。

纵有千般思与念，聆无父母唤儿音。

# 风雨人生

及笄失祜泪偷藏，风雨肩担昼夜忙。

半读半农勤学苦，扶娘搀弟至情长。

注：1960年读初一时父亲突然辞世，学校周日及节假日则参加生产队劳动挣工分帮助家庭艰难度日。

# 挽父母联

春晖恩未报，堂前空忆椿萱貌；

秋雨痛忧添，耳畔难聆叮嘱音。
